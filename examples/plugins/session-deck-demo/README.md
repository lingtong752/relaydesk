# Session Deck Demo

This example plugin shows the minimum frontend plugin shape for RelayDesk:

- local `relaydesk.plugin.json` manifest
- `local_bundle` frontend entry
- host bridge subscription
- host RPC invocation

Install it from RelayDesk Settings -> Plugins using:

- `sourceType`: `local`
- `sourceRef`: `/absolute/path/to/relaydesk/examples/plugins/session-deck-demo`
