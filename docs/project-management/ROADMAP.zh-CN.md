# RelayDesk 路线图

最后更新：2026-03-28

## 1. 当前阶段

当前项目处于 `Phase 4 进行中 - 文件、终端、Git 只读工作台已落地，替身 AI Agent 已具备审批、人工接管和恢复运行基础能力，WebSocket 重连与页面状态回补已接入，Claude 与 Codex 两条真实 Provider 链路已落地，下一步推进检查点与更多 Provider`。

已完成：

- 新仓初始化与 monorepo 结构
- MongoDB 数据连接与基础集合
- JWT 鉴权
- 项目、会话、消息、运行 API
- WebSocket 实时订阅链路
- 登录页、项目页、工作台页
- 替身 AI Agent 启动/停止骨架
- 文件树与基础编辑器
- 浏览器终端工作台
- Git 状态与 diff 工作台
- 替身 AI Agent 最小审批流
- 替身 AI Agent 人工接管与恢复运行
- WebSocket 自动重连与页面状态回补
- Claude 真实 Provider 首条链路
- Codex 真实 Provider 首条链路
- 文档目录体系建立
- 开源治理与基础 CI/测试门禁

- 新功能必须先补对应文档，再进入实现
